HTML is a markup language that is typically used for designing web pages and it stands for Hyper Text Markup Language. Hyper Text Markup Language breaks down and explains how a web page is structured and how things on the web page are displayed. HTML is a series of elements and these elements display headings, paragraphs, links, and more.
